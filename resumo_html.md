# Aprendizados de HTML - Hora de Codar

## O que é HTML?
HTML é uma **Linguagem de Marcação de Hipertexto**. É importante lembrar que ela **não é uma linguagem de programação**, pois não lida com lógica, mas sim com a estruturação de elementos que compõem uma página web.

## Glossário de Tags
- `<h1>` a `<h6>`: Títulos da página, do mais importante ao menos importante.
- `<p>`: Define um parágrafo de texto.
- <a>: Cria links (utiliza o atributo `href` para o destino).
- <img>: Exibe imagens (utiliza `src` para o caminho e `alt` para acessibilidade).

## A Importância da Tag <div>
A tag `<div>` é um container invisível usado para agrupar elementos. Ela ajuda no **aninhamento**, permitindo organizar o código em blocos lógicos para facilitar a estilização posterior.
